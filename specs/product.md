# Product Specification: JOAN MATA Personal Portfolio / CV

## Vision
A premium, high-performance, and interactive digital portfolio that serves as the professional profile and ecosystem hub for Joan Mata Pàrraga. It must reflect a "Senior Engineer" aesthetic, combining deep technical information with a sleek "State-of-the-Art" UI.

## Core Goals
- **Interactive CV**: Modern representation of experience, education, and skills.
- **Project Ecosystem Hub**: Showcase the subdomains and local server infrastructure (`joanmata.com`, `cv.`, `refnotes.`, `f1.`).
- **Multilingual Support**: First-class support for Spanish (ES), Catalan (CA), and English (EN).
- **Security Visibility**: Display security measures (SSL, Cloudflare, Docker) to demonstrate technical competence.

## User Persona
- **Recruiters/Partners**: Looking for high-level technical skills and professional history.
- **Students**: Observing professional standards and teaching resources.
- **Collaborators**: Interested in the infrastructure and shared projects.

## Key Features & Sections
1. **Hero**: Big, bold name with a personal profile description (visible only on the main tab).
2. **Experience**: Timeline or grid of professional roles.
3. **Projects**: Premium interactive grid with modals for technical deep-dives (tech stack, security, github, live links).
4. **Education & Certs**: Academic background and specialized certifications.
5. **Skills**: Visual representation of the tech stack.
6. **Volunteering**: Social commitment and impact.
7. **Contact**: Essential networking links.

## Aesthetic Requirements
- **Dark Mode First**: Deep blacks and slate colors.
- **Accents**: Gradients using Indigo (#6366f1) and Purple (#a855f7).
- **Glassmorphism**: Backdrop blurs and subtle borders for cards and modals.
- **Micro-animations**: Smooth transitions between tabs and modal entry.
